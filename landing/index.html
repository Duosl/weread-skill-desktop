技术架构

现代、轻量、安全

基于 Tauri 2 构建，Rust 后端保证性能与安全，React 前端提供流畅体验。安装包不到 10MB。

R
Rust 后端
API 调用、数据解析、文件导出全部由 Rust 处理。内存安全，零 GC 暂停，启动即用。
R
React 前端
React 19 + TypeScript + Tailwind CSS，类型安全，组件化架构，精致而克制的 UI。
T
Tauri 2
比 Electron 轻 90%。原生窗口，系统级 API 访问，极小的安装包与内存占用。
