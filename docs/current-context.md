## 下一阶段产品原则

- 不把工具扩成微信读书客户端。
- 不为假设中的未来能力增加复杂结构。
- 默认沿用现有 Markdown 导出工作台和笔记工作台结构。
- 阅读报告继续走模板目录和单模板工作台，不把所有模板操作铺在列表页。
- 涉及划线原文、书摘、个人想法时，必须让用户明确知道会读取什么、为什么需要、数据保存在哪里。
- 新功能优先服务“导出可信度、内容可读性、报告复用、外部集成反馈”，而不是补齐 App 里的推荐 / 社区能力。
