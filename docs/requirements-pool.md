## REQ-014 智能体模板原文权限说明优化

- 优先级：P1
- 状态：Todo
- 模块：Report / Privacy / Agent
- 背景：高级报告模板会根据策略读取统计、书架、笔记数量、划线原文和个人想法。涉及原文时需要更清楚地向用户解释数据边界。
- 产品结论：
  - 当前阶段继续使用“必需授权 / 可选授权”两类策略。
  - 必需授权模板未授权时不能生成。
  - 可选授权模板未授权时可以生成，但只能使用低敏数据。
  - 用户自定义要求不能绕过原文授权策略。
- 验收：
  - 前端模板卡片和工作台用用户语言说明数据读取范围。
  - 生成配置摘要明确当前是否会使用划线原文和个人想法。
  - 未授权时 Agent 输入明确要求不要编造划线原文、个人想法、读后感或书籍原文。
  - 必需授权模板未授权时给出可理解原因。
  - 可选授权模板未授权时不写入 `notes.raw.json`。

## REQ-016 Post-MVP 用户反馈驱动优化池

- 优先级：P2
- 状态：Todo
- 模块：Product / Release
- 背景：MVP 已可用，后续新增需求应来自真实用户反馈、复现路径或明确场景。
- 方向：
  - 导出可信度：重导出、跳过原因、失败恢复和结果复核。
  - 内容可读性：Markdown 结构、章节组织、想法与划线关系、Frontmatter 字段说明。
  - 报告复用：基础报告真实数据回归、高级报告隐私授权、输出质量提示。
  - 资料库集成：Obsidian / ima / 本地文件夹的索引友好说明。
- 暂不进入：PDF、在线分享平台、完整书籍详情页、公共书评、推荐发现、在线阅读。
