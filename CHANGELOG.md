# 更新日志

本日志只记录用户可感知的变化，不记录内部重构、依赖调整或验收细节。

## 0.4.5

### 优化

- 自动升级微信读书 Skill 版本号

## 0.4.4

### 优化

- Windows 打包只打 exe，删除无用资源。

## 0.4.3

### 新增

- 书架支持封面墙视图，可在列表和封面墙之间切换。
- 划线和个人想法支持生成分享卡片，并保存为 PNG 图片。
- 阅读报告新增高级模板，可生成通用网页、PPT 风格网页和小红书图文风格网页。
- 支持把导出的 Markdown 笔记同步到指定 ima 知识库。
- 设置页增加匿名安装统计开关和重置入口。

### 优化

- 产品名称和用户可见署名统一为「书迹」。
- 划线/想法分享入口改为更清晰的侧向工作台，预览、样式和保存动作更集中。
- 笔记浏览支持按划线颜色筛选，更容易定位重点内容。
- 阅读报告模板采用目录式选择，生成过程和历史任务更容易追踪。

### 修复

- 改善长内容、分享卡片和报告任务在小窗口下的布局稳定性。
- 优化导出和同步结果说明，减少用户不知道文件或知识库状态的情况。

## 0.4.0

### 新增

- 支持配置微信读书 API Key。
- 支持同步书架、笔记本和阅读统计。
- 支持浏览单本书的划线、个人想法和点评。
- 支持单本或批量导出 Markdown。
- 支持基础阅读统计和基础 HTML 阅读报告。

### 优化

- Markdown 导出包含 Frontmatter，便于 Obsidian 和个人资料库索引。
- 导出页支持真实内容预览，选择单本书时可先查看将要生成的内容。
